# Max Gopp - Graphic Design Portfolio

A dark Victorian-style portfolio website showcasing graphic design work with burgundy jewel tones and elegant typography.

## Features
- Dark Victorian aesthetic with brownish-burgundy color scheme
- Responsive grid layout optimized for vertical poster displays
- Smooth animations and hover effects
- Clean, professional design

## Hosting on GitHub Pages

### Setup Instructions:

1. **Create a new repository** on GitHub
   - Name it something like `portfolio` or `maxgopp.github.io`
   - Make it public

2. **Upload these files** to your repository:
   - `index.html`
   - `BoyWithuke.png`
   - `burnout.png`
   - `sky.png`
   - `escapet.png`
   - `README.md`

3. **Enable GitHub Pages**:
   - Go to your repository Settings
   - Scroll to "Pages" section
   - Under "Source", select "main" branch
   - Click Save

4. **Access your site**:
   - Your portfolio will be live at: `https://[your-username].github.io/[repository-name]`
   - Or `https://[your-username].github.io` if you named it `[your-username].github.io`

### File Structure:
```
portfolio/
├── index.html          # Main website file
├── BoyWithuke.png      # BoyWithUke poster design
├── burnout.png         # Burnout album cover
├── sky.png             # The Sky Is Never Truly Dark poster
├── escapet.png         # Escape poster design
└── README.md           # This file
```

## Customization

To update your portfolio:
- Edit the text in `index.html`
- Replace images with your new designs (keep the same filenames or update the `<img src="">` tags)
- Update your email in the Contact section

## Technologies Used
- HTML5
- CSS3 (with custom properties for theming)
- Google Fonts (Playfair Display, Cormorant Garamond)
